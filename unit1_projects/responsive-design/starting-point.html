Responsive Design and Fluid Layouts

Questions and Answers about the Use of Responsive Design

By: Rob Kramer and Alberto Villacorta

Responsive Design

According to Girl Develop It, Responsive Design is, "Design by thinking of the structure of your website, and how different elements fit into the structure."1 It is used to solve the issue of viewability of a web site on a multitude of different screen types, from mobile browsers to extremely large, flat-panel displays, because creating a different version of a website for every type of screen size is not practical.2 A few of the factors that developers must consider when designing a site are:

  • Screen resolution
  • Browser choice
  • If the browser maximized, or not
  • Accessibility issue

These factors make a "one-size-fits-all" layout virtually impossible these days. Responsive Design gives us the tools to tackle the planning of our site to help meet our objective of the best overall usability across devices.3

Fluid Layouts

Fluid layouts use percentage as a width so they tend to fill the entire width of a browser adjusting to the users set up. It can also eliminate having to horizontally scroll for users. Also, the extra white space between browsers and resolutions tends to be similar.

The developer needs to adjust media and other content with set widths to compensate for different screen resolutions.4

Conclusion: Purpose and Practicality

The purpose of responsive design and fluid layouts is to provide viewablitly and functionality aross a wide range of devices with a large variation in screen size. Although developers have less control over testing for all environments, the use of responsive design provides for better utility of a site over many devices without having to create different sites for every one of them.

Although at times, fluid layouts are not the best solution for a site. It is a beneficial style to implent for the functionality of many sites today. It is practical and we believe its use will only increase as new technologies require it.
